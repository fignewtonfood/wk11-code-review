# _Book Store_
## _Website for bookstore, with new book reviews, users, and exclusive content 10/16/2015_
### By Tim White
## Description
_Users may create account and sign in to access book reviews. List of reviews on left side-bar. Ability to add new review included for reviewer role._

## Credentials
DB name:                        book_store
DB login:                       admin
DB password:                    bookstore

site admin login:               admin
site admin password:            bookstore

reviewer login:                 reviewer
reviewer password:              bookstore

authenticated user login:       user
authenticated user password:    bookstore

## Setup

Clone https://github.com/fignewtonfood/wk11-code-review.git
Start Apache and PHP servers
Import database located in sites/db-backup using phpMyAdmin
Point browser to localhost

## Technologies Used
_This project makes use of: Apache, MySql, PHP, Drupal with Chaos Tools, Features, Views, and StrongArm modules._
